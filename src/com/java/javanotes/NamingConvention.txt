| Type              | Convention     | Example               |
| ----------------- | -------------- | --------------------- |
| **Class**         | PascalCase     | `PaymentProcessor`    |
| **Interface**     | PascalCase     | `Serializable`        |
| **Method**        | camelCase      | `calculateSalary()`   |
| **Variable**      | camelCase      | `employeeAge`         |
| **Constant**      | UPPER\_CASE    | `MAX_RETRY`           |
| **Package**       | lowercase      | `com.company.project` |
| **Generic Types** | Single letter  | `<T>, <E>, <K, V>`    |
| **Test Class**    | ClassName+Test | `EmployeeDetailsTest` |
